## options

<p style={{ fontSize: 18 }}>
  This props is a object to be used for extra customization. <br />
  The options has two property as <strong>toggleContext</strong> and <strong>
    removeCss
  </strong>. <br /> <br />
  If you set toggleContext value except <i>nulish data</i> (undefined and null)
  , it set for sweep button text. <br /> This feature usually is used for some
  component like a toggle component. <br /> <br />
  The removeCss default value is false but if you set true value, it removes all
  css package and you can set your custom styles with className
</p>

```
options= {{
  toggleContext: undefined,
  removeCss: false,
  userSelect: "none" // ser can not mark the text
}}

```

## value-setValue

<p style={{ fontSize: 18 }}>
  The value props is used to set default button and after every click, it is
  changed by buttons value. <br />
  And setValue props get a setState function to update your state after clicking
  on deactive button.
</p>

```jsx
import { useState } from "react";
import { SwitchMultiButton } from "switch-multi-button";

function Component() {
  const [state, setState] = useState("first");

  return (
    <SwitchMultiButton
      value={state}
      setValue={setState}
      buttons={[
        {
          text: "First btn",
          value: "first",
        },
        {
          text: "Second btn",
          value: "second",
        },
      ]}
    />
  );
}
```

## changeTo

<p style={{ fontSize: 18 }}>
  With the changeTo props, you can change the active button by calling the{" "}
  <strong> byValue </strong> or <strong> byIndex </strong> methods.
</p>

<p style={{ fontSize: 18 }}>Bottom code show you how to use changeTo props:</p>

```jsx
import { useState, useRef } from "react";
import { SwitchMultiButton } from "switch-multi-button";

function Component() {
  const [state, setState] = useState("one");
  const changeTo = useRef();

  return (
    <div>
      <SwitchMultiButton
        value={state}
        setValue={setState}
        changeTo={changeTo}
        buttons={[
          {
            text: "first button",
            value: "one",
          },
          {
            text: "second button",
            value: "two",
          },
        ]}
      />

      <button
        onClick={() => {
          changeTo.current.byValue("one");
        }}
      >
        Change active button by value
      </button>

      <button
        onClick={() => {
          changeTo.current.byIndex(2);
        }}
      >
        Change active button by index
      </button>
    </div>
  );
}
```

## buttons

<p style={{ fontSize: 18 }}>
  With buttons props we can create our buttons items. <br /> Simple useage is
  just set <strong> text </strong> and <strong>value</strong> property. <br />
  The <strong>value</strong> property is only required.
</p>

```
buttons = {
  // text => number, string, Component
  text: 'first btn',

  // => number, string
  value: 'first',

  // => button element type ('a', 'p', 'pre', 'section', ...)
  Button: 'div',

  // key => set your custom keys for handle list keys (package handle keys too).
  key: '',

  // className => root element className
  className: '',

  // activeItemClass => use for active item
  activeItemClass: '',

  // activeRootClass => can be customize root style
  // when each item ativate (like a switch component to change background color)
  activeRootClass: '',

  // props => it set for Button item props
  // you can set all of the elemet props with object property
  props: {
    onClick: () => {},
    href: '', // for a tag
    ...
  },
}

```

```
SwitchMultiButton.propTypes = {
  /**
   * root style
   */
  style: PropTypes.object,
  /**
   * set state
   */
  value: PropTypes.oneOfType([PropTypes.string, PropTypes.number]).isRequired,
  /**
   * set your setState callback
   * and
   * you can change active button with setState by value
   */
  setValue: PropTypes.func.isRequired,
  /**
   * each object in array creates a button for you,
   * {
   *  text: show button label or children
   *  value: every button have to be unique value
   *  key: set your unique custom (default: package handle)
   *  Button: item element type (default: 'div'), (custom: 'a', 'p', 'pre', ... etc.)
   *  props: {} can be set all of the element props as object property.  => { onclick: () =>{}, ...}
   * }
   */
  buttons: PropTypes.array.isRequired,
  /**
   * root className
   */
  className: PropTypes.string,
  /**
   * button transition time. 1 equal to 1 seconds
   */
  transition: PropTypes.number,
  /**
   * language direction
   */
  direction: PropTypes.oneOf(["ltr", "rtl"]),
  /**
   * {
   *  toggleContext: undefined,
   *  removeCss: false,
   *  userSelect: 'none'
   * }
   * see document for more info
   */
  options: PropTypes.object,
};

```
