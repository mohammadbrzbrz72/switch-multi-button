# Introduction

<br />

<p style={{ fontSize: 18 }}>
  The <strong style={{ color: "red" }}>switch-multi-button</strong> is a react
  package to create a component for switch to
  <strong> multiple buttons </strong> or <strong> toggle </strong> components with
  a movable motion.

<br />
<br />

## Benefits

<p style={{ fontSize: 18 }}>
  - It works with any element size which have spacing in any responsive page.{" "}
  <br />
  &nbsp; Because it has a automatic computing when device or window size changed.
  <br />
  <br />- In horizontal and vertical direction, it works properly in responsive design(mobile,
  tablet, desktop, ...).
  <br />
  <br />- Compatibility with buttons displacement and direction in responsive design.
  <br />
  <br />- Can be structured like a table with tens and hundreds buttons with a movable
  motion.
</p>
